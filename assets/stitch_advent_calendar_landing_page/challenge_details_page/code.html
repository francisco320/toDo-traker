El Perrito Caliente Perfecto

Ayuda a preparar los mejores perros calientes para la cena de Nochebuena.

Descripción

Ejemplos

Requisitos

¡La familia se ha reunido y el hambre apremia! En Venezuela, la cena de Nochebuena no está completa sin unos buenos perros calientes después de las hallacas. Tu misión es crear un programa que determine la combinación perfecta de ingredientes para el perro caliente ideal, siguiendo las peculiares peticiones de cada miembro de la familia.

El programa debe procesar una lista de ingredientes disponibles y una lista de preferencias y restricciones para devolver el perro caliente perfecto.

Detalles del problema

Recibirás dos argumentos: una lista de ingredientes disponibles y un objeto con las preferencias de la familia. Debes retornar una lista con los ingredientes del perro caliente final.
